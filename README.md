# Lispy (lispy.py)

Небольшой интерпретатор Scheme на Python.

## Схема работы

Интерпретатор работает по принципу read-eval-print loop (REPL):

```
Входная строка -> Парсинг (tokenize, parse) -> AST -> Вычисление (eval) -> Результат -> Вывод (lispstr)
```

- **Парсинг**: Преобразует строку в вложенные списки и атомы.
- **Вычисление**: Выполняет выражения в среде (`Env`), поддерживая переменные, функции и замыкания.
- **Вывод**: Форматирует результаты в Lisp-стиле.

## Быстрый старт
- Запуск REPL:
  ```bash
  python lispy.py
  ```
- Пример:
  ```
  (begin (define r 10) (* pi (* r r)))
  ```

